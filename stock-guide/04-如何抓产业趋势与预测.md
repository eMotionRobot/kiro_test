# 第 4 章 如何用 AI 抓产业趋势与做"预测"

> ⚠️ 本章为方法论科普，**不构成投资建议**。文中行业、公司、数字均为**虚构或示意**。AI 的知识有时间截止点，最新政策与数据需用一手资料核对。

---

## 4.1 先纠正一个观念：什么是"预测"

散户常把"预测"理解成"猜明天涨还是跌"。**这是不可能稳定做到的，也不是本章要教的。**

> 真正有价值的"预测"，是 **趋势研判 + 情景推演**：
> - 判断一个产业未来 **1–3 年** 是处在 **渗透率提升期（成长）** 还是 **饱和/衰退期**；
> - 为**不同情景**（乐观/中性/悲观）各做一套应对，而不是赌单一结果。

类比：你无法预测明天会不会下雨的精确时刻，但你能判断"现在是雨季，出门带伞概率有用"。投资也是研判"季节"，不是猜"某一分钟"。

---

## 4.2 抓产业趋势的五步法（配例子）

下面用一个**虚构的"智能驾驶激光雷达"产业**做示范。

### 第 1 步：梳理产业链（上游—中游—下游）

让 AI 画出链条并标出利润分布：

```
上游（核心元器件）：激光器、芯片、光学元件   → 技术壁垒高、毛利高
中游（雷达本体制造）：激光雷达整机厂           → 竞争激烈、毛利被压缩
下游（应用集成）：整车厂、Robotaxi 运营        → 走量、对成本敏感
```

> **洞察**：利润往往集中在**技术壁垒最高的环节**（这里是上游元器件）。如果某中游整机厂打"价格战"，可能增收不增利（呼应第 3 章 G 公司）。

### 第 2 步：找出 3–5 个核心驱动因素

让 AI 列出推动行业增长的关键变量：

1. **政策**：自动驾驶法规是否放开（强驱动，但有不确定性）
2. **技术**：激光雷达成本能否快速下降（决定能否大规模上车）
3. **需求**：搭载激光雷达的车型销量
4. **替代品**：纯视觉方案（如某些厂商坚持纯摄像头）会不会替代激光雷达
5. **成本**：上游芯片/激光器价格

> **关键**：替代品（纯视觉路线）是这个行业最大的"逻辑风险"——如果纯视觉胜出，整个激光雷达逻辑可能被颠覆。

### 第 3 步：判断生命周期阶段（看渗透率）

```
导入期：渗透率 <5%   → 概念炒作多，业绩兑现少，波动大
成长期：渗透率 5%→40% → 增速最快、最容易出大牛股，但竞争加剧
成熟期：渗透率 >40%   → 增速放缓，看龙头集中度
衰退期：被替代/需求萎缩 → 回避
```

> **例**：假设当前激光雷达在新车的渗透率约 8%，正处在"导入期末→成长期初"，这通常是行业增速最快的阶段，但也是竞争和洗牌最激烈的阶段——**高成长伴随高波动**。

### 第 4 步：多空辩论（避免单边思维）

让 AI 分别扮演"看多方"和"看空方"，各列最强论据：

| 看多方（Bull） | 看空方（Bear） |
| --- | --- |
| 渗透率低、空间大 | 纯视觉路线可能替代，逻辑或被证伪 |
| 成本快速下降，加速上车 | 整机厂价格战，毛利被打到很低 |
| 政策逐步放开 | 政策落地节奏可能远低于预期 |
| 龙头有技术壁垒 | 估值已提前透支多年增长 |

> 这一步的价值：**逼自己看到反面**，防止只听好消息。

### 第 5 步：情景推演（乐观/中性/悲观）

| 情景 | 假设 | 该跟踪的关键指标 |
| --- | --- | --- |
| 乐观 | 渗透率快速到 30%+，龙头胜出 | 季度装机量、车型定点数 |
| 中性 | 稳步渗透但价格战持续 | 龙头毛利率、市占率 |
| 悲观 | 纯视觉替代/政策延后 | 纯视觉车型销量、政策进展 |

> **用法**：不预设哪种情景一定发生，而是**持续跟踪关键指标**，哪种情景的证据在增加，就往哪边调整判断。

## 4.3 信息来源建议（权威优先，AI 辅助消化）

AI 适合帮你**消化和结构化**信息，但**信息来源要权威**：

- 上市公司财报、招股书、业绩说明会纪要
- 行业协会、统计局、监管机构公开数据
- 券商行业研报（注意立场，多看几家对冲偏见）
- 龙头公司高管访谈、产业链调研

> ⚠️ **最关键的提醒**：AI 的训练数据有**时间截止点**，最新的政策、销量、事件**可能不在它的知识里**。务必用最新一手资料核对，别让 AI 用过时数据替你下结论。

---

## 4.4 一个"识别 AI 知识过时"的例子

你问 AI："这个行业最新的补贴政策是什么？"
AI 自信地引用了"某年某政策"。

**正确做法**：去监管机构/官网查最新文件，可能政策早已调整或退坡。
**教训**：凡涉及"最新/今年/最近"的事实，AI 的回答都要打问号，必须用一手信息验证。

## 4.5 本章自查清单

- [ ] 我能说清这个行业的产业链和利润集中在哪一环吗？
- [ ] 推动它增长的 3–5 个核心变量是什么？最大的逻辑风险是什么？
- [ ] 它现在处于生命周期哪个阶段（渗透率多少）？
- [ ] 我有没有听过"看空方"的最强理由？
- [ ] 我跟踪的"关键验证指标"是什么？（而不是只盯股价）

---

## 4.6 可复用 AI 提示词模板

**① 梳理产业链与趋势**
```
请帮我梳理【某行业/赛道】：
1. 上游—中游—下游分别是什么，利润主要集中在哪一环、为什么；
2. 推动行业增长的 3–5 个核心驱动因素；
3. 当前处于生命周期哪个阶段（结合渗透率说明）；
4. 分别以"看多方"和"看空方"列出各自最强的 4 条论据；
5. 给出乐观/中性/悲观三情景，并列出每种情景我应跟踪的关键指标。
特别提醒我：哪些信息可能因你的知识截止时间而过时，需要我用最新资料核对。
```

**② 评估"逻辑风险/替代品"**
```
对于【某行业】，请专门分析它最大的"逻辑被证伪"的风险（如技术替代、政策、需求证伪）：
1. 替代方案/威胁是什么；2. 如果它发生，对该行业的影响有多大；
3. 我应该跟踪哪些先行指标来提前察觉这种风险。
```

### 本章小结

> 抓趋势不是"猜涨跌"，而是研判"产业季节"：梳理产业链→找驱动因素→判断生命周期→多空辩论→情景推演，再持续跟踪关键指标。AI 帮你结构化和当"多空陪练"，但最新信息要靠一手资料。下一章进入心理层面：**如何克服追涨杀跌**。
