# 使用指南 · Buffett Perspective Skill 怎么用

> 一句话：把巴菲特 70 年沉淀下来的「思考方式」装进你的 AI 助手，让它在你做投资、商业、职业决策时，用巴菲特的视角帮你审视一遍。

本指南覆盖：① 它是什么 → ② 怎么安装 → ③ 怎么触发 → ④ 五类典型用法（含完整对话示例） → ⑤ 进阶技巧 → ⑥ 常见问题与边界。

---

## 1. 它到底是什么（先建立正确预期）

这是一个 **Claude Code / Kiro「技能包」（Skill）**，不是一个聊天机器人，也不是一个炒股软件。

- 它的核心是一个 `SKILL.md` 文件 + 一批调研资料（`references/research/`）。
- 当你在对话中说出「触发词」时，AI 会**加载这套思维框架并切换成巴菲特的第一人称视角**来回答你。
- 它输出的是**思考方式、判断逻辑、该问的问题**，而**不是**「现在买入 XX、目标价 YY」这种荐股指令。

> 类比：它不是给你一条鱼，而是借你一副巴菲特的眼镜，让你自己看清那条鱼值不值得钓。

## 2. 安装

### 2.1 用于 Claude Code

```bash
# 克隆到 Claude Code 的 skills 目录
git clone <你的仓库地址>/buffett-perspective ~/.claude/skills/buffett-perspective
```

或者，如果你只是从本仓库（`kiro_test`）取用，把 `buffett-perspective/` 整个目录拷贝到 skills 目录即可：

```bash
cp -r kiro_test/buffett-perspective ~/.claude/skills/buffett-perspective
```

安装后确认目录结构：

```
~/.claude/skills/buffett-perspective/
├── SKILL.md          # 核心：被自动加载的技能定义
├── references/       # 调研证据（按需深度引用）
├── docs/             # 使用/价值/测试文档
└── tests/            # 准确度测试套件
```

Claude Code 会读取 `SKILL.md` 顶部的 frontmatter（`name` + `description`），把它注册为一个可被「触发词」唤起的技能。

### 2.2 用于 Kiro / 其他支持自定义 Steering 的 Agent

把 `SKILL.md` 的内容作为一条 steering / system 规则注入即可。在 Kiro 中，可放入 `.kiro/steering/` 目录，让它对会话生效。

### 2.3 不想安装，临时使用

直接把 `SKILL.md` 的全文粘贴进对话开头，然后说「按上面的设定，用巴菲特的视角回答我」。零安装，但每次都要粘贴。

## 3. 怎么触发 / 怎么退出

### 触发词（说任意一句即可激活）
- 中文：「用巴菲特的视角」「巴菲特会怎么看」「切换到巴菲特」「巴菲特模式」「帮我用巴菲特的角度想想」「如果巴菲特会怎么做」
- English：`Buffett perspective` / `What would Buffett think` / `switch to Buffett`

### 激活后的表现
- AI 用**第一人称「我」**回答，像巴菲特本人在跟你聊天。
- **首次激活**会说一句免责声明（基于公开言论推断、非本人观点），之后不再重复。
- 大量用类比、短句、大白话；不确定就直说「我不知道」。

### 退出角色
- 说「退出」「切回正常」「不用扮演了」即可恢复 AI 的常规模式。

## 4. 五类典型用法（含对话示例）

> 下面的「巴菲特」回答是该 Skill 激活后的**示例输出风格**，用于说明用法，不构成投资建议。

### 用法 A：单只股票 / 一门生意的体检

**你**：用巴菲特的视角看看，一家连续 10 年每年提价但客户不流失的连锁餐饮值得关注吗？

**巴菲特（示例）**：
> 先说结论——这种公司值得我多花点时间，因为"能提价还留得住客户"本身就是一句话讲完的护城河。
>
> 你想想 See's Candies。我们 1972 年买它的时候，很多人觉得贵。但它每年涨价 10%，客户照买不误，因为情人节那天没人会为了省两块钱给丈母娘买二流巧克力。这种**定价权**，就是城堡外面那条又宽又深的护城河。
>
> 不过我得诚实：提价能力只是第一关。我还会问——这门生意 10 年后我说得清它怎么赚钱吗（能力圈）？管理层是把它当自己的孩子养，还是当提款机（管理层诚信）？现在的价格，假设我买下整家公司，划得来吗（安全边际）？这三关过不去，再好的提价能力我也得放进"太难"或者"等等看"的篮子。

**它帮你做了什么**：自动套用了「护城河 → 能力圈 → 管理层 → 估值」的工作流，并给了一个你记得住的类比。

---

### 用法 B：审视你自己已经做了的决策（最高价值场景之一）

**你**：巴菲特模式。我去年追了一只热门 AI 概念股，涨的时候没卖，现在亏 40%，要不要补仓摊低成本？

**巴菲特（示例）**：
> 我先泼盆冷水，再陪你想清楚。
>
> "补仓摊低成本"这个说法本身就有问题——它假设你当初买它是对的，只是买贵了。但真正该问的是：**如果今天我手里没有这只股票，我会用现在的价格买它吗？** 如果答案是"不会"，那补仓不是抄底，是在一个我可能根本不懂的生意上加注。这就像第一次约会很糟糕，你的对策却是再约九次。
>
> 还有一个更扎心的问题：这门生意你说得清它怎么赚钱吗？如果说不清，它对你来说就在能力圈外，亏 40% 不是补仓的理由，是当初就不该上场的信号。沉没成本是沉没的，市场不知道也不在乎你的买入价。

**它帮你做了什么**：把「摊低成本」的思维陷阱用巴菲特的反向提问拆开，引入沉没成本和能力圈视角。

---

### 用法 C：商业 / 管理 / 资本配置决策

**你**：如果巴菲特会怎么做——我们公司账上有一笔闲钱，是该回购股票、发起一笔收购，还是先留着？

**巴菲特（示例）**：
> 这是 CEO 最重要的工作，没有之一——资本配置。我给你一把尺子。
>
> 回购，只有在一种情况下才聪明：你公司的股票卖得比它的内在价值便宜。比内在价值贵还回购，等于用一块钱买八毛钱的东西，那是在毁灭价值，不是创造价值。
>
> 收购最危险，因为有一种我叫"制度性强制力"的东西在作怪——只要你流露出想买的意思，投行和下面的人一定会给你做一份漂亮到无法拒绝的报告。多数收购是为了让 CEO 显得有魄力，不是为了股东。
>
> 留着现金看起来最"没出息"，但我账上趴着三千多亿现金很多年，因为我宁可被嫌"无所作为"，也不愿为了行动而行动。投资没有三振出局，等到那个一英尺的栏杆出现，再跨过去。

---

### 用法 D：人生 / 职业建议

**你**：巴菲特的视角，年轻人到底该不该为了高薪选一份自己没兴趣的工作？

**巴菲特（示例）**：
> 我跟你说个我常说的笑话——为了简历好看去做不喜欢的工作，就像把性生活留到老年再享受，没什么道理。
>
> 严肃地讲：你最值得做的投资是投资你自己。还有，去跟你**敬佩的人**一起工作，这比薪水重要得多。我这辈子最好的决定不是买可口可乐，是娶对了人、跟 Charlie 共事了 60 年。复利不只作用于钱，也作用于你身边的人和你的声誉。

---

### 用法 E：当"魔鬼代言人"帮你做压力测试

**你**：我准备把家庭存款的 70% 投进一个朋友推荐的"稳赚"项目，用巴菲特视角帮我挑毛病。

**巴菲特（示例）**：
> 三个字让我立刻把手从口袋里抽回来——"稳赚"。投资世界里没有"稳赚"，只有"还没暴露的风险"。
>
> 我桌上有张红旗清单，你这事至少踩了两条：一是有人在制造"错过就没了"的紧迫感，可投资从来没有三振出局；二是你说不清它到底靠什么赚钱——如果我没法用一个买菜般简单的类比解释它，那不是它太高深，是我没懂，而我不在不懂的东西上下重注。
>
> 再做个报纸测试：如果这事明天上了报纸头版，一个聪明又刻薄的记者会怎么写你？把家庭 70% 的存款压上去——退潮的时候，你不想成为那个在裸泳的人。

## 5. 进阶技巧

1. **让它显式走工作流**：说「请按你的决策工作流，一步步分析」，它会显式输出「能力圈 → 护城河 → 管理层 → 估值 → 永远测试」五步，适合需要结构化结论的场景。
2. **指定对照案例**：说「用 See's Candies / 可口可乐 / 航空股的教训来对比」，回答会更有锚点。
3. **逼它承认不确定**：问到宏观预测、短期涨跌、加密货币、最新 AI 公司，它应该坦白「我不知道」或放进"太难篮子"。如果它强行预测，说明触发或加载有问题。
4. **组合使用**：先用普通模式让 AI 做事实/数据整理，再「切换到巴菲特」做价值判断，最后「退出」回到执行层面。
5. **配合本仓库的《散户AI炒股实战指南》**：那份指南讲"工作流与纪律"，这个 Skill 补"判断哲学"，两者互补。

## 6. 常见问题与边界（FAQ）

**Q：它能告诉我现在该买哪只股票吗？**
A：不能，也不应该。Skill 内置了硬性护栏：谈原则和思考方式，但不充当持牌投顾，不给"买入 XX、目标价 YY"的指令。这正是巴菲特本人的习惯——他从不在公开场合推荐具体股票。

**Q：它的回答等于巴菲特本人的真实想法吗？**
A：不等于。它是基于公开资料（股东信、采访、传记）的**推断**。对于巴菲特从未公开表态过的新问题（如某家 2026 年新公司），它是在"如果按他的原则推演"，不是"他说过"。详见 `SKILL.md` 的「诚实边界」。

**Q：为什么它有时会说"我不知道"？**
A：这是**特性不是 bug**。坦然承认无知、把超能力圈的东西放进"太难篮子"，正是巴菲特思维最核心的一部分。一个什么都敢预测的"巴菲特"才是假的。

**Q：准确度有保证吗？**
A：我们专门做了准确度测试，方法、用例和结果见 `docs/ACCURACY_TEST.md` 和 `tests/`。你也可以自己跑 `python tests/evaluate.py` 复现评分。

**Q：可以用于中文/英文以外的语言吗？**
A：触发词和示例以中英为主，但模型可用任意语言回应。表达 DNA（类比、大白话、自嘲）是跨语言的。
